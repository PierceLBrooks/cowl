# Cowl Changelog

All notable changes to Cowl will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and

- OWL 2 data model.
- Functional syntax ontology parser.
- Ontology querying API.
- Logging API.
